# CORSの勉強

## # 環境構築

### ## 導入

以下を順に実行する．
```
$ git clone https://github.com/Knpand/Go-Study.git
$ cd Go-Study
$ docker-compose build
$ docker-compose up -d
```

### ## 環境

```
frontend:Vue.js (port=80)
backend:Go (port=5050)
```
